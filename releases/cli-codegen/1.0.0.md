# @damatjs/cli-codegen 1.0.0

## Changed

The `codegen` and `barrel` commands now live in `@damatjs/cli-codegen`. Link
resolution and generated-file augmentation are separate internal units.

Codegen reads resolved package entries/models and writes immutable-package
output under the application rather than inside the package artifact. Model
provider directories are scanned directly, and registry types derive from the
resolved default module entry.

## Action required

Custom composers should import `codegenCliCapability`. The `damat` command
surface is unchanged.
